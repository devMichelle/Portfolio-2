Træks
(Semester project 2)

Træks is an e-commerce website that have both customer-facing and admin sections. The website is populated by a Strapi API provided by Noroff.
The customer-facing section includes product pages and shopping cart, building a checkout and payment system was not a part of this project.
The admin section includes a login, and the functionality of creating, updating and deleting products.
